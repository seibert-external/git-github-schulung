# Git &amp; GitHub – Grundlagen für Einsteiger

Eine Schulungsunterlage für **absolute Einsteiger ohne Entwickler-Hintergrund**
(„Ich bin kein Dev und möchte verstehen, was diese Werkzeuge sind").

Das Material erklärt das Git-Modell und alle wichtigen Begriffe **langsam, in
Alltagssprache und mit einfachen Zeichnungen** – mit Schwerpunkt auf der
GitHub-Weboberfläche und einem praktischen Mitmach-Teil am Ende.

## Inhalt

- **[`git-github-schulung.html`](git-github-schulung.html)** – die komplette
  Schulungsseite. Eine einzige, eigenständige HTML-Datei **ohne externe
  Abhängigkeiten** (funktioniert offline per Doppelklick).

## Verwendung

- **Als Präsentation:** Datei im Browser öffnen und Kapitel für Kapitel
  durchscrollen. Oben gibt es eine Sprung-Navigation.
- **Als Handout:** Die Seite an die Teilnehmer weitergeben – sie können sie in
  Ruhe nachlesen. Über „Drucken" (Strg/Cmd + P) werden Glossar und Spickzettel
  als sauberes Handout ausgegeben; die Trainer-Tipps werden dabei ausgeblendet.

## Themen

Problem ohne Versionsverwaltung · Git ≠ GitHub · Repository · Commit ·
Die drei Bereiche (Arbeitsverzeichnis / Staging / Repository) · Branches · Merge ·
Clone / Push / Pull · Pull Request &amp; Review · Kommandozeile · Mitmach-Teil ·
Quiz · Glossar · Spickzettel.
